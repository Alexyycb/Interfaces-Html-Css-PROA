# Interfaces-Html-Css-PROA
Este repositório reúne as atividades de desenvolvimento de interfaces realizadas no VS Code, utilizando HTML e CSS, propostas pelo PROA e orientadas pela professora Débora, responsável pela Equipe 1 de Web Java.

Os projetos foram desenvolvidos com foco na estruturação semântica do HTML, organização do código, estilização com CSS e aplicação de conceitos fundamentais de UI (Interface do Usuário) e UX (Experiência do Usuário).

Aqui estão registrados exercícios práticos, páginas estáticas, layouts responsivos e desafios técnicos que demonstram minha evolução na construção de interfaces web.

O objetivo deste repositório é documentar meu aprendizado, praticar boas práticas de desenvolvimento front-end e consolidar conhecimentos em estrutura, estilização e organização de projetos.
